# Smart Quiz System 🎯

Smart Quiz System is a Django-based web application that allows users to solve quizzes, calculate scores, and receive recommendation packages based on weak topics.

---

# Features 🚀

- User Registration & Login
- Logout System
- Quiz Solving
- Score Calculation
- Recommendation Package System
- Search Topics
- Pagination
- Responsive UI Design
- Dynamic Django Templates
- Database Models with Relationships

---

# Technologies Used 💻

- Python
- Django
- HTML
- CSS
- SQLite3

---

# Installation ⚙️

Clone the project:

git clone <your-repository-link>

Go into project folder:

cd smart_quiz

Create virtual environment:

python3 -m venv venv

Activate virtual environment:

source venv/bin/activate

Install Django:

pip install django

Run migrations:

python3 manage.py makemigrations
python3 manage.py migrate

Start server:

python3 manage.py runserver
